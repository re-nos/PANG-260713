# FEATURE — Mission 1 난이도 및 규칙

`docs/FEATURES/game_rule.md`에 정의된 공통 게임 룰을 기반으로, Mission 1(첫 스테이지)에 한정된 난이도와 세부 규칙을 정의한다.

## 1. 목적

- 신규 플레이어가 기본 조작(이동, 발사)과 핵심 규칙(풍선 분열, 접촉 시 즉사, 클리어 조건)을 무리 없이 학습할 수 있는 진입용 스테이지를 구성한다.
- 이후 미션(2, 3 ...) 대비 가장 낮은 난이도로 설계한다.

## 2. 난이도 설계 기준

| 항목 | Mission 1 설정 | 비고 |
|---|---|---|
| 풍선 개수 | 1~2개 (최대 크기 기준) | 분열을 포함해도 화면 내 동시 풍선 수는 소수로 유지 |
| 풍선 최대 크기 | Lv.2 ~ Lv.3 | 분열 단계를 짧게 하여 스테이지를 빠르게 클리어 가능하도록 구성 |
| 풍선 이동 속도 | 느림 | 초반 학습을 위해 낙하/바운스 속도를 낮게 설정 |
| 지형 | 평지 위주 (사다리/장애물 없음 또는 최소화) | 이동 관련 학습 부담을 줄임 |
| 파워업 등장 여부 | 등장하지 않음 또는 매우 낮은 확률 | 기본 무기만으로 클리어 가능하도록 설계 |
| 제한 시간 | 없음 (또는 넉넉하게 설정) | 초반 스테이지에서는 시간 압박을 배제 |

## 3. 스테이지 규칙 (Mission 1 한정)

1. 스테이지 시작 시 화면 상단 또는 지정 위치에 최대 크기 풍선 1~2개가 등장한다.
2. 플레이어는 좌우 이동과 위쪽 발사만으로 모든 풍선을 제거해야 한다.
3. 풍선에 접촉 시 생명 1개를 소모하고 스테이지를 재시작한다 (`docs/FEATURES/game_rule.md` 3장 공통 규칙 적용).
4. 모든 풍선(분열된 풍선 포함)이 소멸하면 Mission 1 클리어로 처리한다.
5. 클리어 후에는 결과 화면 또는 다음 미션(향후 확장) 안내로 이동한다. 초기 구현 범위에서는 "클리어" 상태 표시까지만 지원한다.

## 4. 실패 조건 및 재도전

- 생명이 모두 소진되면 게임 오버로 처리하고 메인 화면으로 복귀한다.
- 생명이 남아있는 상태에서 접촉 시, 동일한 Mission 1 스테이지를 처음부터 재시작한다 (풍선 배치 초기화).

## 5. 초기 구현 범위 (MVP)

- Mission 1 단일 스테이지만 구현하며, Mission 2 이후는 범위에서 제외한다.
- 파워업 아이템은 Mission 1에서는 등장시키지 않는 것을 기본값으로 하되, 추후 난이도 조정 시 확률적으로 등장시키는 옵션을 열어둔다.
- 풍선 개수/크기/속도 등 구체적인 수치는 실제 구현 및 플레이테스트를 거쳐 조정될 수 있다.

## 6. 향후 확장 고려사항

- Mission 2 이후 스테이지에서 풍선 개수 증가, 사다리/장애물 추가, 파워업 등장 확률 상향 등을 통해 점진적으로 난이도를 높인다.
- Mission 간 난이도 곡선을 별도 문서(예: `docs/FEATURES/difficulty_curve.md`)로 분리 관리하는 방안을 검토한다.
