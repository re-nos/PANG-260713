# FEATURE — 게임 룰 상세

`docs/PRD.md`에서 정리한 PANG 게임의 개괄적 특징을 바탕으로, 실제 구현 기준이 되는 게임 규칙을 상세화한다.

## 1. 조작

| 입력 | 동작 |
|---|---|
| ← / → (또는 A / D) | 캐릭터 좌우 이동 |
| Space (또는 지정 발사 키) | 위쪽으로 작살(무기) 발사 |
| ↑ / ↓ (사다리 존재 시) | 사다리 오르내림 |

- 캐릭터는 점프할 수 없다. 좌우 이동과 사다리를 통한 상하 이동만 가능하다.
- 작살은 발사 즉시 화면 위쪽 끝(또는 장애물/천장)까지 수직으로 이동하며, 경로상의 모든 풍선과 충돌 판정을 갖는다.
- 동시에 발사 가능한 작살 개수는 기본 1개이며, 파워업(Double Wire) 획득 시 2개로 증가한다.

## 2. 풍선(적) 규칙

### 2.1 크기 및 분열

- 풍선은 크기 단계를 가진다 (예: Lv.4 최대 → Lv.1 최소).
- 풍선이 작살에 맞으면:
  - 현재 크기가 최소 단계(Lv.1)이면 즉시 소멸하고 점수를 획득한다.
  - 그 외의 경우, 한 단계 작은 크기의 풍선 2개로 분열하며 각각 반대 방향으로 튕겨 나간다.
- 풍선은 중력의 영향을 받아 포물선(바운스) 운동을 하며, 바닥/벽/천장에 부딪히면 반사된다.

### 2.2 이동 패턴

- 풍선의 크기가 클수록 바운스 높이가 높고 낙하 속도가 느리며, 작을수록 바운스 주기가 짧고 빠르게 움직인다.
- 풍선은 좌우 벽과 천장에 닿으면 반사되어 반대 방향으로 진행한다.

## 3. 승리/패배 조건

| 조건 | 결과 |
|---|---|
| 화면 내 모든 풍선 소멸 | 스테이지(미션) 클리어 → 다음 미션으로 진행 |
| 캐릭터가 풍선(모든 크기)에 접촉 | 생명 1개 소모, 스테이지 재시작 |
| 생명이 모두 소진된 상태에서 접촉 | 게임 오버 → 메인 화면 또는 결과 화면으로 이동 |

## 4. 생명(라이프) 시스템

- 기본 생명 개수: 3개 (난이도 옵션으로 조정 가능하도록 설계).
- 일정 점수(예: 20,000점, 70,000점 등 임계값)를 달성하면 추가 생명(엑스트라 라이프)을 획득한다. 구체적 임계값은 밸런싱 단계에서 별도로 정의한다.

## 5. 점수 시스템

- 풍선을 터뜨릴 때마다 점수를 획득하며, 풍선 크기가 작을수록(또는 연속 처치 시) 더 높은 점수를 부여하는 방식을 검토한다.
- 스테이지 클리어 시 잔여 시간/생명에 비례한 보너스 점수를 고려할 수 있다 (초기 버전에서는 생략 가능).

## 6. 파워업 아이템

풍선을 터뜨렸을 때 일정 확률로 아이템이 드롭되며, 캐릭터가 접촉하면 즉시 적용된다.

| 아이템 | 효과 | 지속 시간 |
|---|---|---|
| Double Wire | 작살 2줄 동시 발사 | 다음 아이템 획득 전까지 유지 |
| Power Wire | 발사한 작살이 천장/블록에 일정 시간 고정되어 판정 유지 (버튼 연타로 조기 해제 가능) | 제한 시간 또는 버튼 입력 종료 시까지 |
| Vulcan | 연속 고속 사격 가능한 무기로 전환 | 다음 아이템 획득 전까지 유지 |

## 7. 스테이지 종료 및 진행

- 모든 풍선 제거 시 클리어 연출(예: 짧은 대기/이펙트) 후 다음 미션으로 자동 전환한다.
- 초기 구현 범위(MVP)는 Mission 1 단일 스테이지로 한정하며, 이후 미션 확장 시 본 문서의 규칙을 공통 기반으로 재사용한다.
